Pre Check-in

Optimize check-in with online processing and eKYC.

Guests receive a confirmation email with a QR code or Web App link for online check-in. They can complete the check-in process within 24 hours before arrival, reducing the need for physical document photocopying.

  • Features: Online check-in, eKYC data extraction, ID scanning
  • System: eIMS stores and manages check-in details
  • Reports: C06 form export, integration with Vietnam’s ASM system

Output Invoice

Seamless e-invoice integration with Tax Authorities.

An intermediate e-invoice system integrated via API with Tax Authorities and sales software. Supports e-Invoice generation following Circular 78, tax code creation from POS/PMS, and ensures data uniqueness.

  • Compliance: E-invoice follows Circular 78 (T form)
  • Tax Code: Generates M-form tax codes from POS/PMS
  • Management: eIMS Webserver handles issuance, adjustments, and more
  • Integration: Opera, Infor, Infrasys, Simphony, Smile, EZCloud, Oasis
  • Fallback: TFservice Tool for non-API systems
  • Delivery: Send single or batch invoices via email
  • Automation: Issue invoices anytime or submit them at day's end

Intput e-Invoice

Efficiently manage and track all incoming invoices for the hotel.

A centralized system for managing all input invoices from suppliers and service providers, ensuring accuracy, compliance, and seamless integration with financial operations.

  • Invoice Types: Utility bills, supplier invoices, maintenance expenses, and all other input invoices.
  • Integration: Compatible with PMS/POS software (Opera, Simphony, Infrasys, Infor, Smile, Oasis, etc.).
  • Automation: Streamline invoice validation, approval, and payment processes.
  • Compliance: Manage invoices in compliance with Vietnam’s Ministry of Finance regulations.

    • Key Features:

    • Automatically verify and categorize invoices.
    • Attach supporting documents (contracts, receipts, delivery notes, etc.).
    • Provide audit trails and real-time tracking for financial transparency.
    • Ensure secure digital storage for future reference and reporting.

PreBill

PreBill is a web-based application that allows hotel guests to update their invoice details easily by scanning a QR code or accessing a provided link. This ensures accuracy and compliance before the data is transmitted to the General Department of Taxation.

  • Client: Southwest
  • Category: Web Application

Key Features:

  • Guests can update or edit invoice details such as tax code, customer name, ID number, passport, company name, etc.
  • Accessible via QR code or a direct link for a seamless experience.
  • Ensures updates are made within the allowed time frame before tax submission.
  • Enhances data accuracy and reduces errors in the invoicing process.

OFIS API

Seamless integration between Opera PMS and eIMS for automated e-invoicing.

OFIS API enables automatic data transfer from Opera PMS to eIMS for e-invoice generation. The system streamlines the invoicing process by eliminating manual input, ensuring accuracy and compliance with financial regulations.

  • Client: Window
  • Category: Hospitality Technology

Key Features:

  • Automated Data Transfer: When the receptionist clicks "Check Out" in Opera, guest data is automatically transferred to eIMS for e-invoice generation.
  • Real-time E-Invoice Status: The system updates Opera with the latest e-invoice status from eIMS, providing real-time tracking.
  • Seamless Integration: Direct API connection between Opera PMS and eIMS, eliminating the need for manual data entry.
  • Regulatory Compliance: Ensures all invoices comply with Vietnam's Ministry of Finance regulations.
