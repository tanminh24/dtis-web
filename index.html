ID Scan Pre Arrival & C06 Report

The Online Check-in module of eIMS allows hotels to digitize the pre-arrival process while ensuring compliance with the Ministry of Public Security of Vietnam.

Guests receive a confirmation email with a QR code or Web App link for online check-in. They can complete the check-in process within 24 hours before arrival, reducing the need for physical document photocopying.

  • System: eIMS stores and manages check-in details
  • Reports: C06 form export, integration with Vietnam’s ASM system

Key Features:

  • Online Check-in via QR Code or Link Guests receive a booking confirmation email that includes a QR code or link to the check-in WebApp, allowing them to complete check-in within 24 hours before arrival.
  • eKYC and ID Scanning Integration Automatically extracts guest information from scanned identification documents (ID card, passport), minimizing manual data entry and reducing processing time.
  • Real-time Data Synchronization with eIMS All check-in data is securely stored and managed on the eIMS system, accessible by front office staff for tracking, verification, and reporting.
  • C06 Report Generation Automatically generates temporary residence reports in compliance with Form C06 required by the Ministry of Public Security of Vietnam.
  • ASM System Integration Enables uploading of C06 reports to the ASM immigration management platform, supporting full compliance with local authorities.

Output e-invoice Management

An intermediate e-invoice system integrated (API) with Vietnam Tax Authorities and sale softwares

An intermediate e-invoicing system integrated via API with Vietnam’s Tax Authorities and leading PMS/POS software. Fully compliant with Circular 78, Decree 123/2020/NĐ-CP, and Decree 70/2025/NĐ-CP, this module allows for the issuance, adjustment, and management of e-invoices directly from the eIMS Webserver.

Key Features:

  • Compliance: Generate and manage e-invoices in accordance with Circular 78, Decree 123, and Decree 70. Supports full-format e-invoices (T-form) required by the General Department of Taxation (GDT).
  • Tax Code Management: Automatically generate M-form tax codes from POS and PMS systems. Each invoice includes a unique 23-character tax code for tracking and validation.
  • Invoice Management: Manage the full lifecycle of e-invoices including issuance, cancellation, adjustment, and re-issuance through the eIMS Webserver.
  • System Integration: API-ready integration with Opera, Infor, Infrasys, Simphony, Smile, EZCloud, Oasis, and other major hotel management systems.
  • Non-API Support: For systems without API capability, integration is handled via the TFservice tool to ensure uninterrupted operation.
  • Invoice Delivery: Send individual or batch e-invoices to guests via the hotel’s official email domain.
  • Automation Options: Issue invoices in real time or schedule automatic submission to the Tax Authority at the end of each business day.
